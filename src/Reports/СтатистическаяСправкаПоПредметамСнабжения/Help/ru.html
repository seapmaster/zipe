1.1.1. Отчет «Статистическая справка по запрашиваемым заказчиками предметам снабжения за период» формируется на основании параметров, выбранных на странице отображения отчета

Для вывода данных в отчет «Статистическая справка по запрашиваемым заказчиками предметам снабжения за период» необходимо выбрать последовательно значения в следующих полях:

– «Период» – дата начала и дата окончания периода, за который формируется отчет;

– «Страна» – ссылка на элемент справочника, значение выбирается из связанного справочника «ОКСМ»;

- «Проект» – проект корабля, относительно которого формируется отчет. Значение выбирается из списка значений справочника «Проекты кораблей», ограниченного выбранной ранее страной;

- «Корабль» – корабль, относительно которого формируется отчет. Ссылка на элемент справочника, значение выбирается из списка значений каталога кораблей, ограниченного выбранным ранее проектом.
