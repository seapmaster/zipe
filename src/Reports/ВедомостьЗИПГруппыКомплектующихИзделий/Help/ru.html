Отчет «Ведомость ЗИП группы комплектующих изделий (комплектующего изделия) из структуры корабля по комплектующим изделиям и ЗИП» формируется на основании параметров, выбранных на странице отображения отчета.

Для вывода данных в отчет «Ведомость ЗИП группы комплектующих изделий (комплектующего изделия) из структуры корабля по комплектующим изделиям и ЗИП» необходимо выбрать значения параметров в следующих полях:

– «Корабль» – корабль, для которого формируется ведомость ЗИП. Значение – ссылка на элемент связанного каталога кораблей, обязательное поле;

– «Язык представления данных» – язык представления данных по кораблю. Значение – ссылка на элемент связанного справочника «Языки», ограниченного русским языком и языком представления данных, закрепленных за флотом, к которому приписан выбранный корабль. По умолчанию – русский язык;

– «Вид ЗИП» – вид комплекта ЗИП, для которого формируется ведомость. Выбор значения выполняется из списка значений «Бортовой», «Базовый», «Длительной эксплуатации», обязательное поле;

– «Оборудование» – группа оборудования или изделие, для которых формируется ведомость ЗИП. Значение – ссылка на элемент связанного справочника «Комплектующие изделия и ЗИП», обязательное поле.
