Предназначен для мониторинга задач. По умолчанию в отчет попадут все задачи за текущий месяц с истекшим сроком исполнения.

Открывается по команде Задачи, выполненные с нарушением сроков в списке Отчеты по задачам раздела Бизнес-процессы и задачи.

 В отчете задачи сгруппированы по исполнителям. В строки отчета выводится задача, важность, плановая и фактическая дата завершения задачи, указывается количество просроченных дней. Отчет формируется за период. Возможны отборы по Периоду и по Автору.
