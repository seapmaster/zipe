Отчет «Номенклатура запасных частей к комплектующему оборудованию, установленному на кораблях и судах ВМС (форма 2)» формируется на основании параметров, выбранных на странице отображения отчета.

–  «Страна» – страна, относительно которой формируется отчет. Ссылка на элемент справочника, значение выбирается из связанного справочника «ОКСМ», обязательное поле;

– «Проект» – проект корабля, относительно которого формируется отчет. Значение выбирается из раскрывающего списка проектов кораблей, сформированного на основании отобранных данных справочника «Проекты кораблей» с учетом значения, введенного в поле «Страна»;

– «Корабль» – корабль, относительно которого формируется отчет. Ссылка на элемент справочника, значение выбирается из списка значений каталога кораблей, ограниченного выбранным ранее проектом;

– «Заводской номер» – заводской номер корабля, относительно которого формируется отчет, числовое значение вводится с клавиатуры;

– «Система» – корабельная система, относительно которой формируется отчет. Значение выбирается из структуры корабля по КИ и ЗИП в случае, если в поле «Корабль» определен корабль;

– «Изготовитель» – изготовитель оборудования, относительно которого формируется отчет. Ссылка на элемент справочника, значение выбирается из справочника «Организации»;

– «Оборудование» – оборудование, относительно которого формируется отчет. Значение выбирается из структуры корабля по КИ и ЗИП в случае, если в поле «Корабль» определен корабль.
