Для вывода данных в отчет «Перечень поставщиков материалов (форма 9)» необходимо выбрать последовательно значения в следующих полях:

– «Проект» – проект корабля, относительно которого формируется отчет. Значение выбирается из справочника «Проекты кораблей»;

– «Корабль» – корабль, относительно которого формируется отчет. Ссылка на элемент справочника, значение выбирается из списка значений каталога кораблей, ограниченного выбранным ранее проектом;

– «Система» – корабельная система, относительно которой формируется отчет. Значение выбирается из структуры корабля по комплектующим изделиям и ЗИП в случае, если в поле «Корабль» определен корабль;

– «Изделие» – изделие, относительно которого формируется отчет. Значение выбирается из структуры корабля по КИ и ЗИП в случае, если в поле «Корабль» определен корабль.
