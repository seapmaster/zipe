Краткое описание и инструкция по применению

Данные нового корабля заполняются на следующих вкладках:

- «Основное»;

- «Боевые части»;

- «Помещения»;

- «Комплектующие изделия и ЗИП»;

- «Ремонт и модернизация»;

- «Присоединенные файлы».

 

Переход по ссылкам, отличным от ссылки «Основное», возможен только после заполнения обязательных полей данных и сохранения создаваемого элемента нажатием кнопки «Записать» или кнопки «Записать и закрыть», если необходимо записать созданный элемент и закрыть страницу создания элемента.

 

По ссылке «Основное» для нового корабля заполняются следующие поля:

- «Наименование» – символьное значение длиной не более 150 символов вводится с клавиатуры, уникальное в рамках каталога кораблей, обязательное поле;

- «Номер проекта» – ссылка на элемент справочника, значение выбирается из связанного справочника «Проекты кораблей», обязательное поле;

- «Флот» – ссылка на элемент справочника, значение выбирается из связанного справочника «Флоты», обязательное поле;

- «Заводской номер» – символьное значение длиной не более 10 символов вводится с клавиатуры, уникальное в рамках каталога кораблей, обязательное поле;

- «Бортовой номер» – символьное значение длиной не более 10 символов вводится с клавиатуры;

- «Дата постройки» – короткий формат даты, значение вводится с клавиатуры или выбирается из встроенного в поле ввода календаря, обязательное поле;

- «Проектант» – ссылка на элемент справочника «Организации» заполняется автоматически после заполнения поля «Номер проекта»;

- «Строитель» – ссылка на элемент справочника, значение выбирается из связанного справочника «Организации», обязательное поле;

- «Дополнительная информация» – символьное значение неограниченной длины вводится с клавиатуры.

Для создаваемого корабля на данной странице также отображается графическое изображение корабля, в случае наличия соответствующих присоединенных файлов.Также для элемента «Корабль» по ссылке «Основное» заполняются группы «Технические характеристики» и «Вооружение». Для того чтобы развернуть или свернуть данные группы, необходимо нажать левой кнопкой манипулятора типа «мышь» на одноименную ссылку.

 

Группа «Технические характеристики» содержит перечень характеристик корабля и их значений и заполняется данными из связанного справочника «Характеристики кораблей». Для заполнения группы необходимо выполнить следующую последовательность действий:

- нажать кнопку «Добавить» или выбрать одноименный пункт контекстного меню в группе «Технические характеристики», в результате чего создастся строка с номером для ввода характеристики. Номер характеристики создастся автоматически на основе счетчика. Изменение порядка следования характеристик в списке выполняется с помощью кнопок перемещения;

- в поле «Характеристика» открыть связанный справочник «Характеристики кораблей»;

- выбрать необходимую характеристику в открывшейся форме связанного справочника;

- заполнить поле «Значение» для характеристики вводом с клавиатуры или с использованием встроенного калькулятора в поле. Вводимое значение должно быть цифровым длиной не более 12 знаков, также значение может быть дробным – 9 знаков в целой части и 2 знака в дробной;

выбрать значение единицы измерения характеристики из связанного справочника «ОКЕИ»

 

Группа «Вооружение» содержит перечень наименований единиц вооруженной техники корабля с указанием их количества и заполняется данными из связанного справочника «Вооружение кораблей». Для заполнения необходимо выполнить последовательность действий, аналогичную действиям заполнения группы «Технические характеристики»:

- нажать кнопку «Добавить» или выбрать одноименный пункт контекстного меню в группе «Вооружение», в результате создастся строка с номером для ввода вооружения;

- открыть связанный справочник «Вооружение кораблей»;

- выбрать вооружение в открывшейся форме связанного справочника;

- заполнить поле «Количество» вводом с клавиатуры или с использованием встроенного калькулятора в поле;

выбрать значение единицы измерения вооружения из связанного справочника «ОКЕИ».

 

На вкладке «Боевые части» создается структура боевых частей редактируемого корабля.

Форма создания нового элемента структуры «Боевые части (создание)» содержит две ссылки «Основное» и «Присоединенные файлы». Переход по ссылке «Присоединенные файлы» возможен только после заполнения обязательных полей ссылки «Основное»и сохранения создаваемой боевой части нажатием кнопки «Записать» или кнопки «Записать и закрыть», если необходимо записать созданный элемент и закрыть форму создания элемента «Боевые части (создание)».

Для новой боевой части заполняются следующие поля по ссылке «Основное» в форме «Боевые части (создание)»:

- «Краткое наименование» – символьное значение длиной не более 10 символов вводится с клавиатуры, уникальное в рамках корабля, обязательное поле;

- «Полное наименование» – символьное значение длиной не более 100 символов вводится с клавиатуры, уникальное в рамках корабля, обязательное поле;

- «Родитель» – ссылка на элемент списка «Боевые части»;

- «Дополнительная информация» – символьное значение неограниченной длины вводится с клавиатуры.

 

В случае указания значения в поле «Родитель» создаваемая структура является иерархической.

По ссылке «Присоединенные файлы» формы создания (редактирования) боевой части добавляются файлы графического формата (например, «*.jpg»), содержащие дополнительную визуальную информацию по редактируемой боевой части. Управление графической информацией по боевой части аналогично управлению графической информацией по ПС. 

 

На вкладке «Помещения» создается структура помещений редактируемого корабля.

Форма «Помещения (создание)» создания нового элемента структуры «Помещения» имеет вид, представленный на рисунке

.Для нового помещения заполняются следующие поля по ссылке «Основное» в форме «Помещения (создание)».

- «Номер» – символьное значение длиной не более 7 символов вводится с клавиатуры, обязательное поле;

- «Наименование» – символьное значение длиной не более 100 символов вводится с клавиатуры, обязательное поле;

- «Родитель» – ссылка на элемент списка «Помещения», значение выбирается из списка созданных в системе элементов «Помещения»;

«Комментарий» – символьное значение неограниченной длины вводится с клавиатуры

 

На вкладке «Комплектующие изделия и ЗИП» (рис. Рисунок 108) создается структура КИ и ЗИП редактируемого корабля.

Рисунок 108 – Вкладка «Комплектующие изделия и ЗИП»

Структура КИ и ЗИП может быть иерархической. Выбор типа добавляемого элемента (группа, изделие, ЗИП) выполняется в меню кнопки «Добавить», расположенной на вкладке «Комплектующие изделия и ЗИП».

При выборе добавляемого изделия типа «группа» на странице создания новой группы заполняются следующие поля:

- «Корабль» – ссылка на название корабля, для которого формируется структура КИ и ЗИП, автоматически заполняемое поле;

- «Родитель» – ссылка на группу «Комплектующие изделия и ЗИП», значение выбирается из списка созданных ранее в группе изделий для указанного выше корабля;

- «Наименование» – символьное значение длиной не более 150 символов вводится с клавиатуры на русском языке, обязательное поле;

- «Наименование (доп.)» – ссылка на элемент таблицы переводов наименований формируется по группе из таблицы переводов либо значение в поле может быть введено с клавиатуры, отображается в случае отличия языка представления от русского;

- «Заведование» – ссылка на элемент списка «Боевые части», созданного на одноименной вкладке редактируемой структуры корабля;

- «Номер» – символьное значение длиной не более 15 символов вводится с клавиатуры по маске ввода ББББББ-ЦЦЦ-ЦЦЦЦ (Б – буква, Ц – цифра);

- «Обозначение (доп.)» – символьное значение вводится с клавиатуры;

- «Код» – символьное значение формируется автоматически;

- «Входимость» – ссылка на наименование группы в случае формирования вложенной структуры групп КИ и ЗИП, автоматически заполняемое поле;

- «Примечание» – символьное значение длиной не более 150 символов вводится с клавиатуры.

Кнопка «В ниж. рег.» переводит в нижний регистр те значения поля «Наименование (доп.)» или их часть, которые введены в верхнем регистре.

Кнопка «Чертеж» открывает меню управления списком присоединенных файлов редактируемой группы КИ и ЗИП, которое становится доступным только после сохранения создаваемого элемента нажатием кнопки «Записать» или кнопки «Записать и закрыть». В присоединенных файлах может быть прикреплено изображение в графическом формате (например, «*.jpg») в случае, если группа изделий представляет собой общекорабельную систему или общекорабельное устройство.

При отсутствии присоединенных файлов меню кнопки «Чертеж» содержит только один пункт «Добавить».

Для того чтобы добавить новый файл в список присоединенных файлов, необходимо нажать кнопку «Добавить» и стандартными средствами ОС Windows выполнить выбор файла.

При наличии присоединенных файлов меню кнопки «Чертеж» состоит их пунктов «Показать», «Изменить», «Удалить».

Для добавления в структуру корабля изделия типа «изделие» или типа «ЗИП» необходимо выполнить следующую последовательность действий:

- нажать кнопку «Добавить» и в раскрывшемся меню кнопки выбрать пункт «Добавить изделие» или «Добавить ЗИП»;

- выбрать нажатием левой кнопки манипулятора типа «мышь» в открывшемся каталоге ПС необходимое изделие;

- нажать кнопку «Выбрать», в результате чего откроется страница редактирования выбранного изделия, отображающая информацию из каталога ПС;

- ввести дополнительную информацию на открывшейся странице редактирования выбранного изделия;

сохранить внесенные изменения нажатием кнопки «Записать и закрыть» или кнопки «Записать».

 

На странице редактирования изделия типа «изделие» при добавлении в структуру КИ и ЗИП корабля заполняются следующие дополнительные поля на вкладке «Общие сведения»:

- «Наименование (доп.)» – ссылка на элемент таблицы переводов наименований формируется по ПС из таблицы переводов наименований либо значение в поле может быть введено с клавиатуры, отображается в случае отличия языка представления от русского;

- «Количество» – количество изделий данного типа на корабле, цифровое значение длиной не более 3 знаков вводится с клавиатуры;

- «Версия прошивки» – версия прошивки памяти / версия ПО. Символьное значение длиной не более 30 символов вводится с клавиатуры;

- «Дата прошивки» – дата версии прошивки памяти / дата версии ПО. Короткий формат даты, значение вводится с клавиатуры или выбирается из встроенного в поле ввода календаря;

- «Примечание» – символьное значение длиной не более 150 символов вводится с клавиатуры.

Информация, отображаемая в других полях вкладки «Общие сведения» страницы редактирования изделия, является ссылками на соответствующие данные изделия из каталога ПС.

Сохранение внесенных изменений выполняется нажатием кнопки «Записать» или кнопки «Записать и закрыть».

 

После сохранения изменений на странице редактирования изделия становится доступной вкладка «Экземпляры»,на которой отображается список экземпляров КИ и информация об экземпляре.

В группе «Информация об экземпляре» данные по экземпляру КИ заполняются на следующих вкладках:

- «История ТО и Р»;

- «Наработка»;

- «Связанные изделия»;

- «Учет ресурсов и сроков службы».

 

Для нового экземпляра заполняются следующие поля:

- «Судовой номер» – символьное значение  длиной не более 10 символов вводится с клавиатуры, используется при отображении списка, обязательное поле;

- «Заводской номер» – символьное значение  длиной не более 20 символов вводится с клавиатуры, используется при отображении списка;

- «Учетный номер» – номер изделия по справочнику «Юпитер-СТО» символьное значение  длиной не более 10 символов вводится с клавиатуры, используется при отображении списка;

- «Номер позиции» – номер позиции на чертеже системы, символьное значение  длиной не более 5 символов вводится с клавиатуры, используется при отображении списка;

- «Помещение» – помещение,  в котором установлен экземпляр КИ, значение выбирается из структуры корабля по помещениям и является ссылкой на элемент списка «Помещения»;

- «Дата изготовления» – короткий формат даты, значение вводится с клавиатуры или выбирается из встроенного в поле ввода календаря, используется при отображении списка;

- «Дата ввода» – дата ввода в эксплуатацию, короткий формат даты, значение вводится с клавиатуры или выбирается из встроенного в поле ввода календаря;

- «Дополнительная информация» – символьное значение неограниченной длины вводится с клавиатуры.

 

Для новой единицы информации истории ТО и Р заполняются следующие поля:

- «Дата» – дата работ, короткий формат даты, значение вводится с клавиатуры или выбирается из встроенного в поле ввода календаря, обязательное поле;

- «Экземпляр» – ссылка на элемент списка экземпляров КИ, обязательное поле;

- «Вид работ» – ссылка на элемент справочника, значение выбирается из связанного справочника «Виды ремонта», обязательное поле;

- «Трудоемкость работ» – трудоемкость работы в нормо-часах, цифровое значение длиной не более 11 знаков (8 знаков на целую часть и 2 знака на дробную часть), обязательное поле;

- «Трудоемкость сопутствующих работ» – трудоемкость сопутствующих работ в нормо-часах, цифровое значение длиной не более 11 знаков (8 знаков на целую часть и 2 знака на дробную часть);

- «Номер заказа» – номер заводского заказа работ, символьное значение длиной не более 6 символов вводится с клавиатуры;

- «Исполнитель работ» – ссылка на элемент справочника, значение выбирается из связанного справочника «Организации».

 

Для новой единицы информации наработки экземпляров КИ корабля заполняются следующие поля:

- «Дата» – дата ввода значения ресурса, короткий формат даты. Значение вводится с клавиатуры или выбирается из встроенного в поле ввода календаря, или формируется автоматически при занесении значения ресурса через объект «История ТО и Р экземпляров комплектующих изделий», обязательное поле;

- «Экземпляр» – ссылка на элемент списка экземпляров КИ, обязательное поле;

- «Параметр учета ресурса» – ссылка на показатель КИ «Ресурсы и сроки службы», обязательное поле;

- «Признак начала нового межремонтного цикла» – признак фиксации события восстановления ресурса посредством ремонта и начала нового межремонтного цикла;

- «Значение с начала эксплуатации» – значение ресурса на введенную дату с начала эксплуатации, цифровое значение длиной не более 11 знаков (8 знаков на целую часть и 2 знака на дробную) вводится с клавиатуры, обязательное поле;

- «Единица измерения» – единица измерения значения ресурса, ссылка на элемент справочника «ОКЕИ» выбирается вручную или формируется автоматически по параметру учета ресурса показателей КИ, обязательное поле.

 

Для новой единицы информации «Связанные изделия экземпляров комплектующих изделий корабля» заполняются следующие поля:

- «Экземпляр» – ссылка на элемент списка экземпляров КИ, обязательное поле;

- «Связанное изделие» – ссылка на элемент списка экземпляров КИ, обязательное поле;

- «Вид связи» – значение выбирается из раскрывающегося списка значений («Электрическая», «Конструктивная», «Механическая»), обязательное поле.

 

На вкладке «Учет ресурсов и сроков службы». отображается информация по  значениям показателей выбранного экземпляра КИ корабля,  формируемая автоматически на основании данных КИ.

 

На странице редактирования изделия типа «ЗИП»  при добавлении в структуру КИ и ЗИП корабля заполняются следующие дополнительные поля:

- «Тип» – ссылка на элемент справочника «Разделы спецификации ЗИП»;

- «Наименование (доп.)» – ссылка на элемент таблицы переводов наименований формируется по ПС из таблицы переводов наименований либо значение в поле может быть введено с клавиатуры, отображается в случае отличия языка представления от русского;

- «Количество» – количество изделий данного типа, выбранных в качестве составной части или ЗИП в вышестоящем элементе структуры корабля. Цифровое значение длиной не более 6 знаков (3 знака на целую часть, 2 знака на дробную часть) вводится с клавиатуры;

- «Количество в ЗИП Борт» – количество ПС, выбранных в качестве ЗИП в ЗИП на борту корабля. Цифровое значение длиной не более 6 знаков (3 знака на целую часть, 2 знака на дробную часть) вводится с клавиатуры или с использованием встроенного калькулятора в поле;

- «Количество в ЗИП Б» – количество ПС, выбранных в качестве ЗИП в ЗИП на базе. Цифровое значение длиной не более 6 знаков (3 знака на целую часть, 2 знака на дробную часть) вводится с клавиатуры или с использованием встроенного калькулятора в поле;

- «Количество в ЗИП ДЭ» – количество ПС, выбранных в качестве ЗИП в ЗИП длительной эксплуатации. Цифровое значение длиной не более 6 знаков (3 знака на целую часть, 2 знака на дробную часть) вводится с клавиатуры или с использованием встроенного калькулятора в поле;

- «Количество в ЗИП Р» – количество ПС, выбранных в качестве ЗИП в ЗИП ремонтный. Цифровое значение длиной не более 6 знаков (3 знака на целую часть, 2 знака на дробную часть) вводится с клавиатуры или с использованием встроенного калькулятора в поле;

- «Помещение» – ссылка на элемент списка «Помещения» структуры корабля, в котором хранится ПС, входящий в ЗИП на борту;

- «Место хранения» – место хранения ПС в определенном выше помещении, символьное значение длиной не более 150 символов вводится с клавиатуры;

- «Версия прошивки» – версия прошивки памяти / версия ПО. Символьное значение длиной не более 30 символов вводится с клавиатуры;

- «Дата прошивки» – дата версии прошивки памяти / дата версии ПО. Короткий формат даты, значение вводится с клавиатуры или выбирается из встроенного в поле ввода календаря.

Информация, отображаемая в других полях страницы редактирования изделия типа «ЗИП», является ссылками на соответствующие данные изделия из каталога ПС.

 

На вкладке «Ремонт и модернизация» выполняется управление списком работ по проведению ремонтов и модернизации редактируемого корабля. Для нового элемента «Ремонт и модернизация» заполняются следующие поля:

- «Вид ремонта» – ссылка на элемент справочника, значение выбирается из связанного справочника «Виды ремонта», обязательное поле;

- «Исполнитель» – ссылка на элемент справочника, значение выбирается из связанного справочника «Организации», обязательное поле;

- «Дата начала» – короткий формат даты, значение вводится с клавиатуры или выбирается из встроенного в поле ввода календаря, обязательное поле;

- «Дата окончания» – короткий формат даты, значение вводится с клавиатуры или выбирается из встроенного в поле ввода календаря, обязательное поле;

- «Объем работ» – символьное значение неограниченной длины вводится с клавиатуры.

 

На вкладке «Присоединенные файлы» выполняется управление списком присоединенных внешних файлов, содержащих дополнительную графическую информацию по редактируемому кораблю
